# AI Live Personal

AI Live Personal to aplikacja webowa do interakcji z modelami AI z funkcjonalnościami live chat, udostępniania ekranu i analizy zawartości ekranu w czasie rzeczywistym.

## Funkcjonalności

- 💬 Chat z AI
- 🖥️ Udostępnianie ekranu
- 🎙️ Rozpoznawanie mowy (STT)
- 🔊 Synteza mowy (TTS)
- 🧠 Personalizacja AI
- 📸 Automatyczna analiza ekranu

## Instalacja

1. Sklonuj repozytorium
2. Zainstaluj zależności
3. Uruchom aplikację
